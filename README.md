# Sushant Nepal - Education Internship Portfolio

A portfolio website showcasing the education internship experience at Aspire College, focusing on Python Core training.

## About

- **Name:** Sushant Nepal
- **Program:** Bachelor in Technical Education and Information Technology, 7th Semester
- **Institution:** Aspire College, Kathmandu University
- **Location:** Biratnagar, Morang
- **Internship:** Python Core Training at Aspire College

## Features

- Multi-page portfolio website
- Responsive design
- Image gallery with lightbox
- Admin panel for managing gallery images
- No authentication required for admin access

## Pages

1. **Home** - Overview and highlights
2. **About** - Educational background and internship details
3. **Accomplishments** - Timeline of achievements
4. **Gallery** - Photo gallery of teaching moments
5. **Contact** - Contact information and form
6. **Admin** - Gallery management panel

## Admin Panel

Access the admin panel at `admin.html` to:
- Add new images to the gallery
- Edit existing images
- Delete images from the gallery

No authentication is required. Images are stored in browser localStorage.

## How to Use

1. Open `index.html` in a web browser
2. Navigate through the pages using the navigation menu
3. To manage gallery images, go to `admin.html`
4. Add images by providing:
   - Image URL (use image hosting services or direct URLs)
   - Title
   - Description

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage for data persistence

## Customization

- Edit personal information in the HTML files
- Modify colors and styles in `styles.css`
- Add or remove gallery images through the admin panel
- Update content in each page as needed

## Notes

- Images are stored using browser localStorage
- Default sample images are provided using Unsplash
- Replace sample images with actual teaching photos through the admin panel
- The website works offline after initial load
